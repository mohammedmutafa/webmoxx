Privacy Policy

Last updated: July 6, 2025

This Chrome Extension does not collect, store, or transmit any personal or sensitive user data.

What we access

  • Tabs: Used only to close and restore tabs for the panic button feature.
  • Storage: Used locally to save the tab session temporarily.
  • activeTab: Used to run the extension's logic only on the active tab when needed.
  • Scripting: Used to insert or replace content in the current tab.

Third-party Advertising

This extension displays ads provided by a third-party ad network, Adsterra.

Adsterra may collect non-personal information (such as browser type, IP address, etc.) to deliver relevant ads and improve ad performance. We do not control or access this data directly.

For more information, please review Adsterra's Privacy Policy.

Data Collection

We do not collect, log, or share any personal data ourselves.
